# usage  
  
``` terminal
git clone https://github.com/mmmommm/dotfiles.git ~/dotfiles
cd ~/dotfiles
./setup.sh
./deploy.sh
```  
  
## gitが入っていない場合  
  
gitコマンドがあるかを確認  
`git --version`  
  
brewコマンドがあるかを確認  
`brew list`  
  
それぞれ[ここ](https://tracpath.com/bootcamp/git-install-to-mac.html)を見れば全部できるはず
